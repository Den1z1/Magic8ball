Приложение на языке Kotlin - Magic8ball. 
При нажатии на кнопку выдает рандомный ответ в шаре и рандомный текст на кнопке.

Задачи ,которые стояли передо мной во время разработки этого приложения:
1. Создать рандомайзер ответов в виде магического шара. 
2. Написать простенькое приложение для укрепение базового знания Kotlin.
3. Тестирование Coroutines.

Основа приложения была написана в начале моего обучения. Корутины были добавлены позже для теста их возможностей ,они тут по факту не нужны.
